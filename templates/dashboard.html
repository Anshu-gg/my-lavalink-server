Manage your bot's prefix and nickname settings

Command Prefix

The character(s) users type before commands

Example: if prefix is {{ config.prefix }}, users type {{ config.prefix }}help

Nickname Prefix

Automatically added to new members' nicknames

Example: user "anshu" becomes "{{ config.nickname_prefix }} anshu"
